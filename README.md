## üåü Key Features

### 1. üé• 1-on-1 AI Interviewer
- **Dynamic Questioning:** Generates tailored questions based on your Role (e.g., SDE, Data Scientist) and Target Company (Google, TCS, etc.).
- **Speech-to-Text:** Listens to your voice answers in real-time.
- **AI Feedback:** Provides instant analysis on answer quality, relevance, and technical accuracy.
- **Posture Coach:** Uses Computer Vision (MediaPipe) to detect slouching, hand-to-face gestures, and eye contact.

### 2. üó£Ô∏è AI Group Discussion (GD) Simulator
- **Multi-Agent Simulation:** Interacts with AI agents (**Piyush** & **Anjali**) who have distinct personalities (Assertive vs. Calm).
- **Real-Time Debate:** Agents speak audibly using TTS (Text-to-Speech).
- **Smart Intervention:** You can "Jump In" anytime; agents listen, wait, and react to your points.
- **Auto-Evaluation:** Generates a detailed performance report card with scores on logic, fluency, and leadership.

### 3. üìù Aptitude & Coding Test Suite
- **Round 1 (Gated):** 5-40 MCQ questions on Aptitude & Logical Reasoning. (Must score >50% to unlock Round 2).
- **Round 2 (Coding):** Real-world coding problems with an AI-based code evaluator.
- **Detailed Analytics:** Provides a study plan and areas of improvement based on test performance.

### 4. üé® Modern Organic UI
- Designed with a **"Modern Organic Wood"** theme.
- Features earthy tones (Fawn, Terracotta, Mocha) for a stress-free, professional user experience.
- Soft Neo-Brutalism elements for high accessibility.
